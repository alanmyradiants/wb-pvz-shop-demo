---

Запиши факап в CLAUDE.md этого проекта.

1. Если я ещё не описал — задай мне три вопроса:
   - **Что сломалось?** (одной фразой)
   - **Как починили?** (одной фразой)
   - **Что изменить, чтоб не повторить?** (профилактика — env, проверка, мониторинг, документация)

2. Открой `CLAUDE.md` в корне проекта. Найди секцию `## История факапов`. Добавь запись в формате:

   ```
   - **YYYY-MM-DD** — Что сломалось: [...]. Починили: [...]. Профилактика: [...].
   ```

   Дату возьми сегодняшнюю (МСК). Если в секции уже есть записи — добавь новую сверху (новые сверху, старые внизу).

3. Если был полезный кусок кода/команды для починки — добавь его в записи как блок ```bash или ```js.

4. Сделай:
   - `git add CLAUDE.md`
   - `git commit -m "docs: log incident YYYY-MM-DD <короткое описание>"`
   - `git push`

5. Подтверди что записал. Если этот факап раскрывает паттерн (например «WB API всегда отдаёт 429 утром») — предложи добавить отдельный пункт в «Известные проблемы и нюансы» выше.
