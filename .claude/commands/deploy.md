---
description: Закоммитить и запушить текущие изменения с осмысленным сообщением
---

Сделай деплой текущего проекта.

1. Выполни `git status` и `git diff --stat` — покажи мне что меняется
2. Предложи commit message на основе изменений (одной строчкой, формат: `тип: что сделали` — например `fix: 429 backoff на WB API` или `feat: добавил колонку владельца счёта`). Если я скажу другое — используй моё
3. После моего подтверждения выполни:
   - `git add .`
   - `git commit -m "<message>"`
   - `git push`
4. Если обычный `git push` не идёт — используй PAT-паттерн: `git push https://<PAT>@github.com/alanmyradiants/<repo>.git main`. PAT возьми из env-переменной `GITHUB_PAT` или спроси у меня
5. После успешного пуша сообщи:
   - Что закоммитили (короткая выжимка)
   - Railway/GitHub Actions сейчас начнут деплой (если применимо)
   - Где смотреть логи деплоя

Если в репо есть критичные изменения (удаление файлов, изменение env, миграции) — предупреди ДО пуша и попроси подтверждение.
